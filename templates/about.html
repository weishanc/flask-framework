About the Stock Time-Series Demo

This app was built as part of the milestone projects for the Data Incubator 12 day pre-bootcamp course. As someone familiar with R shiny where controlling page layout is extremely difficult, this was actually an informative process and learning experience.

Data for this project comes from the Qandal Wiki dataset. I was unable to use an API call to download data from 2018, so I downloaded 2017-05-01 to 2017-05-30. It also looks like it only grabbed stocks from A to CHGG. I will need to figure out why it didn't pull all tickers even though I did request any specific tickers in the API call. Future work could involve an API call directly in to the app so it doesn't need a flat file.

Example scripts where I worked through the API call and Bokeh plotting can be found on my GitHub page.
